The concept of Chain of Responsibility is applied in this implementation.

 ->
  Enquiry Handler -> Input ( Free Form Text ) -> Based on keyword found in the text enquiry will be delegated to the appropriate team.

An Enquiry Handler is responsible for receiving input in the form of free-form text. The handler analyzes the text and delegates the enquiry to the relevant team based on keywords found within it.

In real-world scenarios, this delegation process can be intricate. It often involves employing NLP or text search algorithms to determine the appropriate handle function."
